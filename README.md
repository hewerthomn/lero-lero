Adicionar Lero-lero
=====================

Clique com o botão direito para adicionar um texto de lero-lero no Google Chrome.

Inspirada na extensão Paste 'Lorem Ipsum...' e no lendário Gerador de Lero-Lero.

[Ver plugin na Chrome Web Store][2]
